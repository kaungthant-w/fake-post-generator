== Description ==
This plugin generates fake posts for testing purposes. It is useful for developers who need to populate their WordPress site with dummy content.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/fake-post-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the plugin by navigating to the 'Fake Post Generator' menu in the WordPress admin.

== Frequently Asked Questions ==
= How do I use this plugin? =
After activating the plugin, go to the 'Fake Post Generator' menu in the WordPress admin and follow the instructions to generate fake posts.

= Can I customize the generated posts? =
Yes, you can customize the number of posts, post titles, and content through the plugin settings.
